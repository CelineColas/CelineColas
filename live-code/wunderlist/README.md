# Wunderlist - TODO Manager

## Interface expectations

$ ruby app.rb
Things you can do:
1. List all the tasks
2. Create a new task
...
What's your choice?
> 1

[x] buy croissants
[ ] buy coffee
[ ] build an application

## Features

- list all the tasks
- add a new task
- remove a task
- mark a task as done

## Task

Properties:

- name
- done (boolean, default: false)

opt. properties:

- description
- deadline
- priority
- ...

Behavior:

- mark as done
